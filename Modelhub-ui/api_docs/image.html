GPT Image 2 可用

基于 GPT 的图像生成模型，支持多比例输出与参考图输入

生成图像

POST https://api.wuyinkeji.com/api/async/image_gpt

请求 Header

名称值
Authorization接口密钥
Content-Typeapplication/json

请求参数

参数必填类型说明
prompt 必填 string 图像描述提示词
size 可选 string 输出图像比例，默认 auto
可选值：auto / 1:1 / 3:2 / 2:3 / 16:9 / 9:16 / 4:3 / 3:4 / 21:9 / 9:21 / 1:3 / 3:1 / 2:1 / 1:2
urls 可选 array 参考图片 URL 数组，支持多张
示例：["https://example.com/a.jpg"]

请求示例

// POST https://api.wuyinkeji.com/api/async/image_gpt?key=你的密钥 { "prompt": "一只穿着宇航服的柴犬在月球上散步", "size": "16:9", "urls": ["https://example.com/ref.jpg"] }

返回示例

{ "code": 200, "msg": "成功", "data": { "id": "image_4d39239e-776a-4cbd-a8eb-e2d9b4816829", "count": 10 }, "exec_time": 0.29, "ip": "119.6.176.239" }

查询任务状态

GET https://api.wuyinkeji.com/api/async/detail?id=任务ID

请求参数

参数必填类型说明
id 必填 string 生成接口返回的任务 ID，如 image_4d39239e-...

返回参数

字段类型说明
data.statusint任务状态码（见下方）
data.messagestring错误信息（失败时返回）
data.resultobject/array生成结果，包含图片 URL
状态码说明：
0 = 初始化 1 = 进行中 2 = 成功 3 = 失败
任务为异步处理，提交后需轮询此接口直到 status 为 2 或 3。建议轮询间隔 3 秒。

返回示例（成功）

{ "code": 200, "msg": "成功", "data": { "status": 2, "result": "https://cdn.example.com/generated.jpg" }, "exec_time": 12.5 }
