Helmet 安全头
设置 X-Content-Type-Options、X-Frame-Options 等 HTTP 安全响应头，防止 XSS 和点击劫持
请求体大小限制
限制请求体最大 1MB，防止超大 Payload 耗尽服务器内存
IP 封禁系统
支持手动封禁和自动封禁（10分钟内违规10次自动拉黑30分钟），封禁列表持久化存储
攻击模式检测
自动识别 SQL 注入、XSS 攻击、路径遍历、命令注入，检测到严重攻击立即封禁 IP
全局降速
同一 IP 前 30 次/分钟正常响应，之后每次请求延迟递增 200ms，最大延迟 5 秒
并发连接限制
限制每个 IP 最多 10 个并发连接，超出直接返回 429，防止多线程并发攻击
API 频率限制
通用接口 60次/分钟，登录注册 10次/15分钟，管理后台 30次/分钟
攻击日志记录
所有可疑请求自动记录到日志文件，包括 IP、UA、攻击类型、请求详情
扫描器检测
识别 sqlmap、nikto、nmap 等常见攻击工具的 User-Agent，记录并累计违规
